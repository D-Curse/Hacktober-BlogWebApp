
# GDSC-Web Hacktober Contribution Repo.

How to run this project ?

- For frontend :

    1. Change the directory - ``` cd frontend```
    2. Install the dependencies - ``` npm install```
    3. Run the application - ``` npm run dev```

- For backend : 
    1. ``` cd backend ```
    2. ``` npm install ```
    3. ``` node server.js```
